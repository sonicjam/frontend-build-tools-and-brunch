What Brunch does

Yeoman and Brunch cover same field

  • They are more like framework than Grunt.js
  • Main differences between Yeoman/Brunch and Grunt.js:
    • Script modularization (AMD/CommonJS)
    • Boilerplate
    • Scaffolding

Difference between Yeoman and Brunch

AMD CommonJS Boilerplate Scaffolding Package manager
Yeoman ✓ ✓ ✓ ✓
Brunch ✓ ✓ ✓ ✓
  • Yeoman is built on Grunt.js, Require.js, and Twitter Bower
  • Brunch is built with own built-in functions

Why I choose Brunch

  • Supports both of AMD and CommonJS modularization
  • Super simple Boilerplate compared to Yeoman
  • Super simple Scaffolding compared to Yeoman
  • Twitter Bower is like a reinventing wheel of NPM
